# pyja
PYthon Java Analysis

# Installation
## Install dependencies
`./install_deps.sh`

## Install the library
`./install.sh`


# Dependency types
  * `d`: DEPENDENCY
  * `s`: SUPERTYPE
  * `t`: TYPE_PARAMETER
  * `f`: FIELD
  * `p`: METHOD_PARAM
  * `r`: RETURN_TYPE
  * `v`: VARIABLE

# Class types
  * `c`: Controller
  * `r`: Repository
  * `e`: Entity
  * `s`: Service
